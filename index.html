What's so great about canvas?


  • Interactivity

    Canvas is fully interactive. It can respond to a user’s actions by listening for keyboard, mouse, or touch events.

  • Animation

    Every object drawn on the canvas can be animated. This allows for all levels of animations to be created

  • Flexibility

    Developers can create just about anything using canvas -- lines, shapes, text, images, video, audio

  • Browser/Platform Support

    HTML5 Canvas is supported by all major browsers

  • Popularity

    Canvas popularity is rapidly and steadily growing so there is no shortage of resources available.

  • It's a web standard

    Unlike Flash and Silverlight, Canvas is open technology that’s part of HTML5

  • Develop once, run everywhere

    HTML5 Canvas offers great portability. Once created, Unlike Flash and Silverlight, a canvas application can run almost anywhere
